使用Github+hexo搭建个人博客

(一)安装使用软件：

1.node.js下载地址-http://nodejs.cn

2.Git下载地址-https://git-scm.com/

安装好后可以输入以下检测版本号看是否安装成功

(二)注册Github账号以及建立仓库：https://github.com/

如图注册一个github账号新建一个项目仓库New repository，如下所示：

要创建一个和你用户名相同的仓库，后面加.github.io，只有这样，将来要部署到GitHub page的时候，才会被识别，也就是http://sduzjh.github.io，我的用户名是`sduzjh`

(三)安装配置hexo

你可以先创建一个文件夹myblog，用来存放自己的博客文件

在该目录下右键点击Git Bash Here，打开git的控制台窗口

输入npm install -g hexo-cli安装Hexo

接下来初始化一下hexo,即初始化我们的网站，进入我们主题根目录Hexo，输入hexo init初始化文件夹

然后，接着输入npm install安装必备的组件

最后结果如下

输入hexo g生成静态网页，然后输入hexo s打开本地服务器

可看到效果如图所示

(五)生成SSH添加到GitHub

右键打开git bash，然后输入下面命令

git config –global user.name “sduzjh”
git config –global user.email “202000130222@mail.sdu.edu.cn“

输入ssh-keygen -t rsa -C "202000130222@mail.sdu.edu.cn"找到key放进GitHub账户里

将显示内容部署到仓库里

(六)修改_config.yml内的文件为自己仓库号将hexo部署到GitHub

修改内容如下图所示

(七)写文章发布文章

首先在博客根目录下右键打开git bash，安装一个扩展npm i hexo-deployer-git。

然后输入hexo new post "我的第一篇博客"，新建一篇文章。

hexo g生成静态网页，然后输入hexo s可以本地预览效果，最后输入hexo d上传到github上。这时打开你的https://sduzjh.github.io/主页就能看到发布的文章了

(八)总结

  1. 注意建立GitHub仓库时仓库名要和账户名一致并添加github.io来保证能够网页直接访问到该仓库
  2. 注意在_config.yml的修改将hexo和GitHub连接起来的时候要修改branch: master和

repository: git@github.com:sduzjh/sduzjh.github.io.git为自己的仓库名来保证顺利连接

  1. 在检查GitHub是否与本地连接时可以输入ssh -T git@github.com，出现自身用户名来保证成功连接
